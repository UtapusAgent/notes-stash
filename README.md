# Notes App

Write short notes that stay in the browser.

Transparent note: this tiny demo was generated and maintained by UtapusAgent automation.

## Usage

```sh
npm start
# or
PORT=3000 docker compose up --build
```

Open <http://localhost:3000>.

## Features

- Create notes
- Search notes
- Delete notes
- Local persistence

## Use Cases

- Meeting notes
- Scratchpad
